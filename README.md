# Handtracking 3D Control

ハンドトラッキングを使用して3D空間内のオブジェクトを物理的に操作できるWebアプリケーションです。
MediaPipeを使用して手の動きを検出し、Three.jsとCannon-esで物理演算を適用した3D空間に反映させています。

## 特徴
- **リアルタイムハンドトラッキング**: ウェブカメラを通じて手の動きを遅延なく検出します。
- **物理インタラクション**: オブジェクトをつかむ、動かす、投げるといった動作が物理法則に従って行われます。
- **3Dレンダリング**: リッチな影とライティングを含む3D環境。

## 必要要件
- Node.js (推奨 v18以上)
- ウェブカメラ付きのPC/Mac

## インストール
プロジェクトのディレクトリで以下のコマンドを実行し、依存関係をインストールしてください。

```bash
npm install
```

## 実行方法
開発サーバーを起動します。

```bash
npm run dev
```

ターミナルに表示されるURL（例: `http://localhost:5173/`）をブラウザで開いてください。
初回アクセス時にカメラの使用許可を求められますので、「許可」を選択してください。

## 操作方法
1. **手の認識**: カメラに手をかざすと、手の人差し指と親指の中心に「赤い球体」が表示されます。これがあなたのカーソルです。
2. **つかむ (Grab)**: オブジェクト（箱や球）にカーソルを近づけ、**人差し指と親指をつまむ（ピンチ）**動作をしてください。
   - つかむとカーソルが「緑色」に変わります。
3. **投げる (Throw)**: つかんだまま手を素早く動かし、指をパッと開くと、勢いよく物体を投げることができます。
